Soportar gradientes con múltiples cambios en IE9 (usando SVG).

Añade una clase "gradient" a todos tus elementos a los que les quieras aplicar un gradiente
y añade estas líneas a tu HTML completar el soporte IE9:

<!--[if gte IE 9]
<style type="text/css">
.gradient {
filter: none;
}
</style>
<![endif]-->
